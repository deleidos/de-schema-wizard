The DigitalEdge Schema Wizard is one of the tools in the DigitalEdge 3.0 Suite. It is used by a data engineer to create or modify schemas. A DigitalEdge schema is intended to be a unified schema for an enterprise system. Such a schema is produced by analyzing data in one or more data samples. Schemas are used by other tools in the suite to include the DigitalEdge Ingest Pipeline and the VizWiz tool.

 

Schema Wizard creates a schema by a discovery process while analyzing sample data. The data may be structured or unstructured and its content may come in various forms such as: comma separated values, JSON, XML, plain text, and documents. Additionally, multiple instances of the data may come in compressed file formats. The data engineer can collect the samples as files prior to using Schema Wizard.
