Application Features

Hierarchical View

This is a feature of the application that allows the user to explore deep into the data sample that is being built into a schema. You can explore the hierarchy of a data sample by clicking the expand button in order to view the structure and hierarchy of the sample. Once completed click the to collapse the view. You will encounter the hierarchical view when creating a schema or looking at data samples/schemas in the catalog,.

User Interface Tour Feature

Throughout the Schema Wizard there are little icons like this . These tour buses are informational tours that are spread across various pages in order to give a detailed tour of each page and it's features.
