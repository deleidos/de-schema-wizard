Authentication

Schema Wizard has demo authentication functionality as of version 3.0.0-beta3. The username and password is publicly available, so any information available to the user should be considered publicly available. Full authentication functionality is expected in future releases.

If you would like to have access to this please contact your system administrator, or Leidos, Inc.
