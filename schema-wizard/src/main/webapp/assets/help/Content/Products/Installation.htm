Installation

Requirements -

  • Linux environment
  • AppArmor
  • Docker 1.11
  • Docker Compose 1.7.1

    • Install docker-compose and clone our* repository. Then, from the project root directory run:

    sudo apparmor_parser -r -W sw-script-profile

    sudo docker-compose up

     

    We recommend you start with the example files in this project.
